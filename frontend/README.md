# Interface Utilisateur : Partie Frontend (WoodPrint)

> Tenez-vous prêt. Ce répertoire est réservé à l'application cliente frontend. Le développement de tous les composants React/Vue, les assets UI, et la symphonie visuelle démarreront strictement **après l'achèvement de l'architecture Backend**. 

### Dépendance Architecturale
Le frontend WoodPrint se comportera de manière dynamique basée sur l'exposition de données JSONB (comme le *schema de personnalisation*) de la base de données. Il est donc impératif d'avoir le cœur de traitement Backend entièrement opérationnel pour orchestrer l'interface utilisateur.
